Run Step

The Run step is the fourth phase within a turn. It is preceded by the Setup step.

At this point, the current user can play .SCRIPT cards from their hand, and issue commands via Idle .PRG and .DAEMON cards in play, as long as have either control or ownership of the card. During this step, other users can also issue commands, always starting from current user in a stack-like fashion (last commands to be executed are the first to be resolved).
